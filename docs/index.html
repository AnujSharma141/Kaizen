Kaizen

Anime Data API

Kaizen is an Open Source API which provides Anime data. Senpai is made with GraphQL and provides flexible endpoints as per the data need. The data is solely taken from myanimewatchlist.com and crawled through web.

Note : Kaizen is NOT a commercial API
