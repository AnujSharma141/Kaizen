Introduction

As Kaizen is built over GraphQL, it is advised to have a basic knowledge of how GraphQL client works with different frontend framewrks. HowtoGraphQL is a great source to learn GraphQL.
After setting up the GraphQL Client, use the given address as the source url.

ADDRESS : https://apikaizen.herokuapp.com/api

Schema

Queries

Now that you might have learn about schema. Have a look at the available queries.

Search

Provide a list of anime matching with the keyword string.

search(key: String!): [Unit]!

query {
search(key : "Hunter x Hunter"){
name
link
}
}

Latest

Provide a list of anime which are currnty airing.

new: [Item!]!

query {
new {
name
link
image
rating
}
}

Popular

Provide a list of anime which are popular.

popular: [Item!]!

query {
popular {
name
link
image
rating
}
}

Rated

Provide a list of highest rated anime.

rated: [Item!]!

query {
rated {
name
link
image
rating
}
}

Details

Provides data of anime of which LINK is requested by.

detail(link: String!): Set!

query {
detail(link : "https://myanimelist.net/anime/
37521/Vinland_Saga"){
name
episodes
genre
description
image
rating
trailer
}
}

Map

Provides data of anime of which NAME is requested by.

map(name: String!): Set!

query {
map(name : "Jujutsu Kaisen"){
name
episodes
genre
description
image
rating
trailer
}
}
